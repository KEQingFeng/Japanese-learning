# 日语学习 AI 资源库 — 提示词・Agent工具流・网站・软件

## 📖 项目简介

这是一个**持续更新的日语学习资源集合**，专注于以下四类内容：

- **提示词（Prompts）**：可用于 ChatGPT、Claude、Gemini、本地开源模型等大语言模型的日语学习专用提示词，涵盖语法讲解、会话练习、作文批改、词汇记忆等场景。
- **Agent 工具流（Agent Toolchains）**：基于 LangChain/LiteLLM 等框架构建的日语学习自动化工作流，如自动生成Anki卡片、间隔复习提醒、错误分析报告等。
- **推荐网站**：经过筛选的高质量日语学习网站（新闻阅读、听力、辞典、社区等）。
- **推荐软件**：桌面端、移动端、浏览器插件等实用工具。

项目的核心理念是**收集、整理、优化**，让学习者无需从零摸索，直接获得经过社区验证的资源和配置。

## 🎯 为什么要做这个项目？

现有的日语学习开源项目多侧重单一功能（语法整理、单词库、辞典等），而**AI时代的日语学习资源**存在以下缺口：

1. **提示词分散且质量参差**：网上很多“日语学习Prompt”未经系统整理，效果难以保证。
2. **Agent工作流缺乏共享**：很多人写了自己的自动化脚本来辅助学习，但缺乏平台共享和协作优化。
3. **优质资源发现成本高**：新网站、新软件层出不穷，需要社区筛选和评价。

本项目希望通过众包方式，汇聚日语学习者和AI爱好者的智慧，建立一个**可持续更新的资源索引库**。

## 📂 资源分类

### 1. 提示词（Prompts）

每个提示词包含：
- 适用场景（如：初级语法讲解、N2读解训练、模拟面试）
- 目标模型/平台（如：ChatGPT 4o、Claude 3.5、本地Llama 3）
- 效果示例（输入/输出截图或文字展示）
- 使用注意事项

### 2. Agent 工具流（Agent Toolchains）

每个工具流包含：
- 功能描述（如：每日自动生成长文阅读+生词提取+造句练习）
- 技术栈（LangChain / LangGraph / AutoGen 等）
- 配置文件或简易代码
- 部署说明

### 3. 学习网站

按类型分类：新闻（NHK Easy News、News Web Easy）、播客、词典（Weblio、goo辞書）、语法参考（日本語の例文）、听力（まるごと）、社区（HiNative、LangCorrect）、考试备考（JLPT相关资料）等。

### 4. 学习软件

包括：Anki插件、浏览器扩展（如Yomitan、Rikai-kun）、iOS/Android应用、桌面工具（如金田先生）等。

## 🔄 更新机制

- 项目将**每周或每两周**集中更新一次，合并社区提交。
- 所有资源均经过至少两名维护者的初步审核，确保质量。
- 欢迎通过 Issue 提交新资源建议，或直接提 PR 添加。

## 🤝 如何参与？

你可以通过以下方式贡献：

- **提交新资源**：在 Issue 中填写资源名称、类型、链接及简短说明。
- **评价已有资源**：在对应目录下的 `.md` 文件中添加你的使用感受或改进建议。
- **优化提示词**：如果你改进了某个 Prompt 并测试有效，欢迎提交更新版本。
- **分享工作流**：如果你写了自动化的学习脚本，请把配置或代码分享出来。

我们尤其需要**日语母语者或中高级学习者**帮助验证提示词效果。
